# Samgraha Help

## Purpose

Welcome to the Samgraha help documentation. This is the complete reference for installing, configuring, and using Samgraha — the knowledge compilation and audit tool for engineering documentation.

## Content

Samgraha compiles structured markdown documentation into a searchable SQLite knowledge database, audits documents against defined standards, and serves the knowledge via CLI and MCP (Model Context Protocol) for AI-assisted development workflows.

This help is organized into:

- **Getting Started** — Installation, initialization, environment setup, first project walkthrough
- **Concepts** — Architecture and terminology: repositories, workspaces, knowledge DB, registry, standards, domains, audit
- **Commands** — Complete CLI reference for every Samgraha command
- **Configuration** — `samgraha.toml` reference covering every configuration section
- **Documentation Guide** — How to write documentation for each Samgraha standard domain
- **Audit Guide** — How the audit system works, deterministic and semantic checks, reports, gates
- **Search Guide** — How to search compiled knowledge, filtering, pagination, cross-repo queries
- **Multi-Repo Guide** — Working with multiple repositories: dependencies, interests, workspaces
- **MCP Guide** — Setting up the Model Context Protocol server for Claude, OpenCode, Codex CLI
- **Build Guide** — Building, releasing, packaging, and distributing Samgraha knowledge
- **Tutorials** — Step-by-step walkthroughs for common workflows
- **Troubleshooting** — Solutions for common issues
- **FAQ** — Frequently asked questions
