# Tutorial: Building Your First Knowledge System

## Purpose

Step-by-step guide to creating a knowledge system that provides validation, reporting, and scaffolding scripts for samgraha.

## Prerequisites

- Samgraha installed and initialized (`samgraha init`)
- Python 3.8+ (or any scripting language of your choice)
- A repository with documentation to validate

## Step 1: Create the System Directory

```bash
mkdir -p my-system/scripts
mkdir -p my-system/templates
```

## Step 2: Create system.yaml

```yaml
id: my-system
name: "My Knowledge System"
version: "0.1.0"
description: "Custom validation for my project documentation"

capabilities:
  validate:
    script: scripts/validate.py
    async: false
  report:
    script: scripts/report.py
    async: false
  scaffold:
    script: scripts/scaffold.py
    async: false

domains:
  - feature
  - architecture
```

## Step 3: Write the Validate Script

Create `scripts/validate.py`:

```python
#!/usr/bin/env python3
import argparse
import json
import sys
from pathlib import Path

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo-root", required=True)
    parser.add_argument("--in", required=True, dest="input_file")
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    # Read input
    with open(args.input_file) as f:
        input_data = json.load(f)

    target = input_data.get("target", {})
    doc_path = target.get("document_path", "")
    check_ids = target.get("check_ids", [])

    # Validate
    findings = []
    full_path = Path(args.repo_root) / doc_path

    if not full_path.exists():
        findings.append({
            "criterion_id": "SYS-001",
            "severity": "error",
            "message": f"Document not found: {doc_path}"
        })

    # Add more checks as needed...

    # Write output
    output = {
        "status": "pass" if not findings else "fail",
        "score": 100.0 if not findings else 0.0,
        "findings": findings,
        "metadata": {"checks_run": len(check_ids)}
    }

    with open(args.out, "w") as f:
        json.dump(output, f, indent=2)

if __name__ == "__main__":
    main()
```

## Step 4: Write the Report Script

Create `scripts/report.py`:

```python
#!/usr/bin/env python3
import argparse
import json
from pathlib import Path

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo-root", required=True)
    parser.add_argument("--in", required=True, dest="input_file")
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    with open(args.input_file) as f:
        input_data = json.load(f)

    # Generate report
    report_path = Path(args.repo_root) / "docs" / "report" / "my-report.md"
    report_path.parent.mkdir(parents=True, exist_ok=True)

    report_content = f"""# Validation Report

Generated by My Knowledge System

## Summary

- Domain: {input_data.get('target', {}).get('domain', 'unknown')}
- Status: Generated
"""
    report_path.write_text(report_content)

    output = {
        "status": "pass",
        "report_path": str(report_path),
        "metadata": {}
    }

    with open(args.out, "w") as f:
        json.dump(output, f, indent=2)

if __name__ == "__main__":
    main()
```

## Step 5: Register the System

```bash
samgraha standards register --path ./my-system
```

## Step 6: Test the System

```bash
# Run validation via MCP
# (through samgraha's MCP interface)
run_system_validate(system_id="my-system", domain="feature")
```

## Step 7: Verify Scripts Work

Check that scripts return valid JSON with the expected schema:

```bash
# Manual test
python my-system/scripts/validate.py \
  --repo-root . \
  --in /tmp/test-input.json \
  --out /tmp/test-output.json

cat /tmp/test-output.json
```

## What You Learned

- Knowledge systems are directories with `system.yaml` and scripts
- Scripts follow the `--repo-root`, `--in`, `--out` contract
- Systems are registered with `samgraha standards register`
- Samgraha discovers and invokes system scripts via capability dispatch
